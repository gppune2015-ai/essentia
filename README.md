# PDF Flipbook

This repository contains a simple HTML/CSS/JS flipbook that loads a PDF directly from Google Drive using PDF.js.

## Files
- `index.html` — main viewer
- `styles.css` — styling
- `script.js` — PDF loading + basic flipbook logic
- `README.md` — documentation

## How to Use
1. Upload all files to a GitHub repository.
2. Enable GitHub Pages in Settings → Pages.
3. Ensure your Google Drive PDF is shared as **Anyone with the link → Viewer**.
4. Update the `DRIVE_FILE_ID` inside `script.js`.

Your flipbook will open at:
`https://yourusername.github.io/yourrepo/`
